## Troubleshooting

### Q: I typed a brand name but don't see the "Create" option
**A:** The brand might already exist with different casing. Try:
- Type exactly as it appears in the list
- Or select the existing brand from the filtered list

### Q: The brand creation is slow
**A:** This usually takes < 2 seconds. If it's taking longer:
- Check your internet connection
- Try again (might be a temporary server issue)
- Contact support if it persists

### Q: I created a brand but it's not saving with my deal
**A:** 
- Make sure the brand is selected in the dropdown (should be highlighted)
- Check that you filled in all required fields (*marked with *)
- Try creating the deal again

### Q: I accidentally created a duplicate brand
**A:** Contact support - they can merge duplicates or delete if needed
